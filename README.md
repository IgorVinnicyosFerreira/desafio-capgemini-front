# desafio-capgemini-front

## Instalação das bibliotecas node

```
npm install
```

### Variaveis de ambiente

Crie um arquivo .env na raiz do projeto e copie o valor do .env.example para ele

### Inicie o servidor local

```
npm run serve
```
